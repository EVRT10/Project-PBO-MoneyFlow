![Swing](https://img.shields.io/badge/Swing-GUI-blue?style=for-the-badge)

**MoneyFlow** is a modern, lightweight, and user-friendly Personal Finance Management desktop application built with Java Swing. It helps you track your daily income and expenses with beautiful 3D charts and an intuitive user interface. 

This project was developed as a Final Project for the Object-Oriented Programming (PBO) course.

---

## ✨ Features

- **💰 Transaction Tracking**: Easily record your daily income and expenses.
- **📊 3D Interactive Charts**: Visualize your cash flow with stunning 3D Pie Charts and Bar Charts.
- **📂 Categorization**: Organize your transactions with customizable categories and fun emojis.
- **💾 Local Storage**: Your data is securely saved locally in CSV format.
- **🎨 Modern UI**: Clean, responsive, and aesthetic user interface built from scratch using Java Swing.

## 🚀 Getting Started

### Prerequisites
- **Java Development Kit (JDK) 11** or higher.

### Installation & Run
1. Clone the repository:
   ```bash
   git clone https://github.com/EVRT10/Project-PBO-MoneyFlow.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Project-PBO-MoneyFlow
   ```
3. Run the Build Script (Windows):
   ```bash
   build_jar.bat
   ```
4. Run the Application:
   Double click the generated `MoneyFlow.jar` or run it via terminal:
   ```bash
   java -jar MoneyFlow.jar
   ```

## 📦 Download Installer (Windows)

Don't want to build it from source? You can download the ready-to-use Windows Installer (`.exe`) directly from the **[Releases](../../releases)** page! Just download, install, and start managing your money.
